# QAチェックリスト（実機レビュー用）

## クリティカルパス
- [ ] 起動 → Start → 質問 → 結果 → マップ起動 → 戻る が途切れない
- [ ] 途中中断しても復帰できる（または明確に最初からになる）

## クラッシュ耐性
- [ ] 5択連打しても落ちない
- [ ] 戻る連打しても落ちない
- [ ] 位置情報拒否でも落ちない
- [ ] アプリをバックグラウンド→復帰しても破綻しない（任意）

## UX
- [ ] 1問1秒で回答できる（ボタンが押しやすい）
- [ ] 進捗が分かる（例: 3/12）
- [ ] 質問文が長すぎない・理解しやすい
- [ ] 結果のTop3が見やすい（確率と理由がある）

## 推論の妥当性
- [ ] こってり寄り回答→こってり系ジャンルが上がる
- [ ] さっぱり寄り回答→さっぱり系ジャンルが上がる
- [ ] UNKNOWN連打でも破綻しない（確率が極端化しない）
- [ ] 同じ質問が同一セッションで再出題されない

## 学習（M5以降）
- [ ] 「これにする」→次回の結果が少しだけ寄る
- [ ] 学習が暴れない（すぐ固定化しない）
- [ ] リセットで完全に戻る

## レイアウト（iPhone実機）
- [ ] 小さい端末でもボタンが画面外に出ない
- [ ] 文字が切れない（質問文、ジャンル名）
- [ ] SafeArea対応
