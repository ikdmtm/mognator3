# プライバシーポリシー / Privacy Policy

**最終更新日: 2026年1月30日**

## モグネイター（Mognator）プライバシーポリシー

### 1. 収集する情報

モグネイターは、サービス提供のために以下の情報を収集します：

#### 1.1 位置情報
- **目的**: 近くの飲食店を検索し、最適な店舗を提案するため
- **収集方法**: アプリ使用時に、ユーザーの同意を得てデバイスの位置情報を取得
- **保存**: 位置情報は一時的に使用されるのみで、サーバーに保存されません

#### 1.2 使用データ
- **目的**: アプリの機能改善とパーソナライズされた推薦を提供するため
- **内容**: 
  - 質問への回答履歴
  - 選択したジャンルと店舗
  - アプリの使用パターン
- **保存**: すべてのデータはデバイス内にローカル保存され、外部には送信されません

### 2. 情報の使用目的

収集した情報は以下の目的でのみ使用されます：

- 近くの飲食店を検索し、表示する
- ユーザーの好みに基づいて店舗を推薦する
- アプリの機能を改善する
- ユーザーエクスペリエンスを向上させる

### 3. 情報の共有

モグネイターは、以下の場合を除き、ユーザーの個人情報を第三者と共有しません：

- **Google Places API**: 店舗情報の取得のため、位置情報をGoogle Places APIに送信します
- **法的要請**: 法律で義務付けられている場合

### 4. データの保存とセキュリティ

- **ローカルストレージ**: 使用データ（質問への回答、学習履歴）はすべてデバイス内に暗号化されて保存されます
- **サーバー保存なし**: 個人を特定できる情報はサーバーに保存されません
- **削除**: 設定画面から「学習データをリセット」でいつでもデータを削除できます

### 5. 第三者サービス

モグネイターは以下の第三者サービスを使用しています：

#### 5.1 Google Places API
- **目的**: 店舗情報の取得
- **送信データ**: 位置情報、検索キーワード
- **プライバシーポリシー**: https://policies.google.com/privacy

### 6. 子供のプライバシー

モグネイターは13歳未満の子供を対象としていません。13歳未満のユーザーから意図的に個人情報を収集することはありません。

### 7. データの権利

ユーザーには以下の権利があります：

- **アクセス権**: 保存されているデータを確認する権利
- **削除権**: 設定画面からデータを削除する権利
- **異議申し立て権**: データの使用に異議を申し立てる権利

### 8. プライバシーポリシーの変更

本プライバシーポリシーは、法律やサービスの変更に応じて更新される場合があります。重要な変更がある場合は、アプリ内で通知します。

### 9. お問い合わせ

プライバシーに関するご質問やご懸念がある場合は、以下までご連絡ください：

**メール**: privacy@mognator.example.com  
**サポートページ**: https://mognator.example.com/support

# Privacy Policy (English)

**Last Updated: January 30, 2026**

## Mognator Privacy Policy

### 1. Information We Collect

#### 1.1 Location Information
- **Purpose**: To search for nearby restaurants and suggest optimal venues
- **Collection**: Device location with user consent during app use
- **Storage**: Location data is used temporarily and not stored on servers

#### 1.2 Usage Data
- **Purpose**: To improve app functionality and provide personalized recommendations
- **Content**:
  - Question answer history
  - Selected genres and restaurants
  - App usage patterns
- **Storage**: All data is stored locally on device and not transmitted externally

### 2. Use of Information

Collected information is used only for:
- Searching and displaying nearby restaurants
- Recommending restaurants based on user preferences
- Improving app features
- Enhancing user experience

### 3. Information Sharing

Mognator does not share user information with third parties except:
- **Google Places API**: Location data is sent to Google Places API for restaurant information
- **Legal Requirements**: When required by law

### 4. Data Storage and Security

- **Local Storage**: Usage data is encrypted and stored on device
- **No Server Storage**: No personally identifiable information is stored on servers
- **Deletion**: Users can delete data anytime from settings

### 5. Third-Party Services

#### 5.1 Google Places API
- **Purpose**: Restaurant information retrieval
- **Data Sent**: Location, search keywords
- **Privacy Policy**: https://policies.google.com/privacy

### 6. Children's Privacy

Mognator is not intended for children under 13. We do not knowingly collect information from children under 13.

### 7. Your Rights

Users have the right to:
- Access stored data
- Delete data from settings
- Object to data usage

### 8. Changes to Privacy Policy

This policy may be updated to reflect legal or service changes. Significant changes will be notified in-app.

### 9. Contact Us

For privacy questions or concerns:

**Email**: privacy@mognator.example.com  
**Support**: https://mognator.example.com/support
